## ğŸ“Š How It Works

### Breathing Exercise Flow
1. User selects "Start Breathing Exercise"
2. Three.js initializes particle system
3. Visual/audio cues guide through inhale (4s) â†’ hold (7s) â†’ exhale (8s)
4. Particles expand/contract in sync with breathing phases
5. Cycle repeats for user-defined duration
6. Optional anxiety level logging post-exercise

### AI Affirmation Generation
1. User rates current anxiety level (1-10 slider)
2. Frontend sends level to `/generate-affirmation` endpoint
3. Flask backend constructs GPT-4 prompt with anxiety context
4. OpenAI generates personalized, calming affirmation
5. Message displayed with gentle fade-in animation
6. User can regenerate or save affirmation

### Anxiety Tracking Visualization
1. User logs anxiety levels over time via `/log-anxiety`
2. SQLite stores entries with timestamps
3. Frontend fetches data from `/get-anxiety-data`
4. Chart.js renders line graph showing mood trends
5. Users can identify patterns, triggers, and progress
